Taller 3 — Árboles de Búsqueda (versión simple y completa)

Estructura del proyecto:
  ArbolBinario.h   -> BST simple
  ArbolAVL.h       -> AVL con rotaciones
  main.cpp         -> lee archivos, mide tiempo, compara, imprime tabla y guarda resultados.csv
  operaciones.txt  -> ejemplo

Compilar:
  g++ -std=c++11 -O2 main.cpp -o app

Ejecutar:
  ./app operaciones.txt
  ./app archivo1.txt archivo2.txt archivo3.txt
